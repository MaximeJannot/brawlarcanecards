# Brawl Arcane Cards
This project offers digital cards to play [Brawl Arcane 28](https://brettfp.itch.io/brawl-arcane-28).


## Technical presentation
The project is based on [Jekyll](https://jekyllrb.com/). It is a static site.
No use of cookies.
